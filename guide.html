Guide to trade on tnbCrow Discord Server

Buy TNBC

  1. Join tnbCrow discord server.
  2. Check the available advertisements on #sell-orders channel.
  3. Buy TNBC from the advertisement using adv buy id: ID amount: AMOUNT command. Private channel is created within the discord server.
  4. Send the payment to seller's payment method.
  5. Wait for the seller to release escrow.
  6. Check your new balance using /balance command.
  7. Withdraw TNBC to your wallet using /withdraw tnbc command.

Youtube Tutorial

Sell TNBC

  1. Join tnbCrow discord server.
  2. Use the /deposit tnbc command to deposit TNBC into your crow bot account.
  3. Use /payment method add command to add the payment method you'd like to sell your TNBC with.
  4. Create an advertisement with /adv create command outlining your sell price and amount.
  5. Wait for the buyer to buy from your advertisement.
  6. If the buyer's decides to buy, a private channel is created within the server to discuss the payment details.
  7. Wait for the buyer to send you the payment.
  8. Once received, release the escrow using /escrow release command.
    9. Warning: Never release escrow before verifying that you have received payment.
